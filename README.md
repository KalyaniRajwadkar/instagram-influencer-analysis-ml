# 📊 Instagram Influencer Analysis & Prediction (ML Project)

## 🧠 Overview
This project focuses on analyzing Instagram influencer data to uncover engagement patterns and predict influence scores using machine learning techniques. It involves complete data preprocessing, exploratory data analysis (EDA), feature engineering, and predictive modeling on real-world social media data.

---

## 🎯 Objective
- Analyze influencer performance based on engagement metrics  
- Identify key factors affecting influence score  
- Build a predictive model for influence score  
- Generate insights for influencer marketing strategies  

---

## 📁 Dataset
The dataset contains information about top Instagram influencers with the following features:
- Rank  
- Channel Name  
- Influence Score  
- Posts  
- Followers  
- Average Likes  
- 60-Day Engagement Rate  
- New Post Average Likes  
- Total Likes  
- Country  

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

---

## 🔄 Workflow

### 1. Data Preprocessing
- Removed duplicates and handled missing values  
- Converted real-world formats (K, M, B, %) into numeric values  
- Cleaned and standardized dataset for analysis  

### 2. Exploratory Data Analysis (EDA)
- Analyzed relationship between followers and engagement  
- Distribution of influence score  
- Country-wise influencer analysis  
- Engagement trends and patterns  

### 3. Feature Engineering
- Created features like:
  - Like-to-Follower Ratio  
  - Post-to-Follower Ratio  
  - Engagement-based metrics  

### 4. Model Building
- Applied Linear Regression and Random Forest models  
- Trained model to predict influence score  
- Evaluated using performance metrics  

---

## 📊 Key Insights
- Engagement rate is not directly proportional to follower count  
- Certain countries dominate influencer presence  
- Feature engineering improves prediction accuracy  
- Random Forest performs better for this dataset  

---

## 📈 Model Output
- Predicted influence score  
- Feature importance analysis  
- Actual vs Predicted comparison  

---

## 📌 Conclusion
This project demonstrates how data analysis and machine learning can be used to evaluate influencer performance and predict their impact, providing valuable insights for data-driven marketing decisions.
